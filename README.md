# Herbstferien-Urlaub — Familienziele im Vergleich

Interaktive Übersicht von sechs sonnensicheren Familienzielen für die **Berliner Herbstferien 2026 (19.–31. Oktober)** — mit Gesamtpreis für die Familie (All-Inclusive **und** Selbstverpfleger, inkl. Flug ab Berlin), erwartbarem Oktober-Wetter und Familien-Highlights.

Familie als Grundlage: **2 Erwachsene + Kind (8 J.) + Kind (5 J.)**, 7 Nächte.

Ziele: Türkei (Antalya), Ägypten (Hurghada), Griechenland (Kreta), Mallorca, Andalusien (Costa del Sol), Portugal (Algarve).

## Fertige Seite ansehen

Einfach `dist/index.html` per Doppelklick im Browser öffnen — die Seite ist self-contained (alle Styles, Skripte und Grafiken inline), es ist kein Server nötig.

## Selbst bearbeiten / neu bauen

Voraussetzung: Node.js 18.17+.

```bash
npm install
npm run dev      # lokaler Entwicklungsserver
npm run build    # erzeugt dist/index.html
```

Alle Inhalte (Ziele, Preise, Wetter) stehen als Datenarray oben in `src/pages/index.astro` — dort lassen sich Werte ergänzen oder korrigieren, danach `npm run build`.

## Hinweis zu den Werten

Preise sind realistische Richtwerte (Stand Juni 2026) für die gesamte Familie, 7 Nächte inkl. Flug ab Berlin im Herbstferien-Zeitraum. „AI" = All-Inclusive, „Selbstverpfleger" = nur Flug + Unterkunft. Tatsächliche Preise hängen stark von Hotel, Buchungszeitpunkt und Verfügbarkeit ab. Wetterangaben sind langjährige Klima-Mittelwerte für die zweite Oktoberhälfte, keine Vorhersage.
